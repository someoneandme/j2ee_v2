ibatis在实际的开发中非常实用，主要原因是：

1. Java没有类似python的'''原始字符串的表达方式，
所以在java代码中写sql时，如果sql太长，就得用记号+来换行，写得很不舒服，
特别是当有'"\等特殊符号时，也麻烦。
而ibatis的xml文件中，可以很方便地写sql语句

2. SQL查询条件中，根据参数来确定要不要加入where，或者怎样加入where条件时，
ibatis就显得非常有用。例如，根据参数是非提供，确定要不要加where；或者把or改成union方式等等
都不需要改到java代码，把sql的编写和优化内聚在xml文件中。

3. 数据库字段和java bean字段的映射关系，这个虽然不是特别重要，
但是在没有hibernate或spring jdbctemplate的情况下，那就很好用了。

============================================
因此，实际开发中，可以在xml很随意地写SQL语句，一般都可以随时嵌入sql或<isNotEmpty>的标签语句
